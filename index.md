# Welcome!

This is the documentation for [Cebola project](https://github.com/orgs/novAIcariusQ/projects/1) - a catalogue for local shops and their customers with AI integration, full shop, product and order control and more.

---

This documentation serves as a knowledge base for **APIs**, **databases** and other essential information, so all team members can understand how everything works to avoid any problems when collaborating.

*If you have something to share*, [go ahead and create a page](https://github.com/novAIcariusQ/Cebola-devdocs)! Main branch isn't protected, so you are free to make changes to it, or you can create a Pull Request and assign [me](https://github.com/infgotoinf) for review, or review it yourself.

Additionaly there is "<u>Edit this page on Github.</u>" button on the **bottom of every page**, so *if you want to fix a typo or add something* to the existing documentation, you can do it right on Github.

# Welcome!

This is documentation for [Cebola project](https://github.com/orgs/novAIcariusQ/projects/1) - a catalogue for local shops and their customers with AI integration, full shop, product and orders control and many more.

---

This documentation exists as a knowlega base to document **APIs**, **Databases** and everything else important, so all team members can understand how everything work, so we don't have any troubles working together.

So *if you have something to share*, [don't be shy, create a page](https://github.com/novAIcariusQ/Cebola-devdocs)! Main branch isn't protected so you are free to push in it, or you can create a Pull Request and assign [me](https://github.com/infgotoinf) for a review, or review it yourself.

Additionaly there is "<u>Edit this page on Github.</u>" button on the **bottom of every page**, so *if you wanna fix a typo or add something* to the docs, you can do it right on Github.
